Centos7 下 LNMP

发表于 2020-02-24 分类于 服务器

宝塔一键快速安装

https://www.bt.cn/bbs/thread-19376-1-1.html

记得保存宝塔用的账号和密码

PHP 配置

安装的PHP版本可能跟WEB不一样
即 php -v 显示的可能会跟 phpinfo() 显示的不一样
此处要配置php-fpm

只保留一种版本

yum -y remove php*

安装所需版本

此处为7.2

php-fpm 配置

出现了 file not found时
vim /etc/php-fpm.d/www.conf

php-fpm 命令

systemctl enable php-fpm
systemctl start php-fpm
systemctl stop php-fpm
systemctl restart php-fpm
装好之后记得开启

Nginx 配置

nginx.conf

cd /etc/nginx/conf.d/xxxxx.conf

Nginx 命令

$ sudo systemctl enable nginx
$ sudo systemctl start nginx
$ sudo systemctl restart nginx
$ sudo systemctl reload nginx
记得重启

配置域名

vim /etc/hosts
