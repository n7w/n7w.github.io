发表于 2020-05-13 更新于 2020-08-20

对 0….n-1 共 $n$ 个点组成的图，一共有 $m$ 条边

存储

邻接矩阵：
vector<vector<int>> graph(n, vector<int>(n, 0))
graph[a][b]：a -> b 的权值

不能开太大，比如 N = 1e4：

无权值存储：
vector<int> graph[N]

邻接表： PII : pair
unordered_map<int, list<<PII>> > graph
graph[a]： 与 a 连接的所有点
graph[a][i].first：a 与 b 连接
graph[a][i].second：a -> b 的权值

遍历

时间复杂度 $O(n + m)$

DFS

BFS
