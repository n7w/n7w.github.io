287. 寻找重复数

Floyd 判环算法（龟兔赛跑算法）

  1. 设定快慢指针 slow，fast 作为起点
  2. 循环： slow 走一步，fast 走两步
  3. 若存在环则终究会相遇 此时 fast 肯定在环里面
  4. slow = 起点
  5. 循环： slow 和 fast 只走一步
  6. 由数学证明可知再次相遇的点为环的起点

128. 最长连续序列

求数组中能组成连续递增的子序列长度

  • 关键是判断 x 的下一个数 x+1 是否存在

  • set 存储法

    每个数字都只访问了一次，故时间复杂度为 O(n)

  • 并查集
    此法相当于模拟了一遍组合过程

    时间复杂度会略微大于 O(n)，但仍属于 O(n) 范围内

126. 单词接龙 II

给出一堆单词，求 a 变换到 b 的最短路径，每次只能改变一个字母，且改变后仍为单词表中的单词

  • 单源最短路径问题，用朴素 dijkstra 会 TLE
  • 由于知道起点和结果，故可用 bfs 搜索
  • 先建无向图，将 string 映射为 int
  • 需要保存路径，可用 dfs 回溯，有 dalao 提供思路为开 queue<vector<int>>， bfs 遍历过程直接就存储好
  • 判断最短路径根据 dijkstra 的做法来更新
