基础算法

快速排序

对 arr[] 进行排序

  • 任意选一个数 x，可为 arr[l]、arr[r]、arr[l + r >> 1]
  • 左右指针 i、j 遍历
  • 一次遍历结束后有 arr[....] < x < arr[....]
  • 分治递归

归并排序

  • 分治递归
  • 设置 temp[] ，将排序结果存储在 temp[] 中
  • 按 l....r 的顺序将 temp[] 的数据转回原数组 arr[]

递归处理方法

  • 设置截止条件
  • 设置变量来修改截止条件
  • 递归
  • 不去考虑到底是怎样递归的，直接思考递归完后会得到怎样的结果

二分查找

  • 确定查找函数 check()，确定目标区间 [l,r] （都是闭区间）
  • 目标 x 满足分割后的 [l, x], [x, r] ，一边全满足 check() 一边都不满足 关键点是mid 处于左区间还是右区间
