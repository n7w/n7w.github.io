LRU 和 LFU

发表于 2020-09-07 更新于 2020-09-09

Least Recently Used (LRU)

最近最久使用算法
满足查询 $O(1)$，插入 $O(1)$，删除 $O(1)$
实现方式：LinkedHashMap，即双向链表 + 哈希表

Least Frequently Used (LFU)

最近最少使用算法
满足查询 $O(1)$，插入 $O(1)$，删除 $O(1)$
实现方式：DLinkList<DLinkList>，即以双向链表为节点的双向链表

双向列表

c++ STL 可使用 list<Node*>::iterator

手写链表：画图模拟
双向链表：定义头尾指针，头尾指针是空的
c++ STL 可使用 list<Node*>

LinkedHashMap (LRU)

c++ STL 可使用 unordered_map<int, list<int>::iterator >
实现了双向链表的查询 $O(1)$

DLinkList (LFU)

以双向链表为节点的双向链表

LRU

实际操作只有一条链表，哈希表只是映射

LFU

对单独的每一条双向链表，满足 front() 是最久未使用的， back() 是最近使用的
对 DLinkList<DLinkList> 来说，满足 front() 是频率最小的双向链表
